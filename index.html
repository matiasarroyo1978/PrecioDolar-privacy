Política de Privacidad – Extensión “PrecioDolar - Cotización dólar Argentina”

Última actualización: 2025-XX-XX

1. Introducción

La extensión PrecioDolar - Cotización dólar Argentina proporciona información pública sobre cotizaciones del dólar. Esta extensión no recopila ni transmite datos personales de los usuarios.

2. Datos que recopilamos

La extensión NO recopila:

  • Información personal identificable
  • Direcciones IP
  • Cookies o identificadores del dispositivo
  • Información financiera del usuario

3. Permisos utilizados

Para su funcionamiento, la extensión solicita permisos mínimos:

  • storage: para guardar preferencias locales del usuario (tema, configuración de alertas, tipo de cambio favorito). Los datos quedan en el dispositivo y no se transmiten a servidores externos.
  • notifications: para enviar alertas opcionales sobre cambios de cotización cuando el usuario habilita esa función.
  • host permissions: acceso a las APIs públicas que proveen cotizaciones (por ejemplo https://dolarapi.com/ u otras URLs públicas). La extensión solo realiza peticiones de lectura para mostrar valores actualizados.

4. Uso de servicios externos

La extensión puede consultar APIs públicas para obtener cotizaciones. No se envía información personal a esos servicios; solo se consultan datos públicos.

5. Transferencia y seguridad

No compartimos, vendemos ni comercializamos datos de usuarios. No almacenamos información personal en servidores externos.
